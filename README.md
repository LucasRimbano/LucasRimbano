## 🚀 Proyectos destacados

### 🌐 Portfolio Web

Portfolio personal desarrollado con HTML, CSS y Bootstrap, orientado a mostrar mis proyectos, servicios y trabajos de desarrollo web.

### 💧 Agua Potable - Análisis de datos y redes neuronales

Proyecto basado en un dataset real sobre potabilidad del agua.  
Trabajé con análisis estadístico, limpieza de datos, tratamiento de valores faltantes, boxplots, correlación de Pearson, normalización de datos y entrenamiento de redes neuronales para predecir si el agua es potable o no.

### 📊 Gestor de operaciones de trading

Aplicación desarrollada en Python para registrar compras, ventas y operaciones abiertas/cerradas de criptomonedas.  
El sistema calcula PNL realizado, cantidad restante, costo proporcional y genera reportes en Excel con resumen por moneda.

### 🎮 Juego en Python
